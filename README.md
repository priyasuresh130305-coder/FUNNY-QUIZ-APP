# Funny Quiz - Android Quiz Application

## Project Overview

Funny Quiz is an Android quiz application developed using HTML, CSS, and JavaScript and packaged as an Android application (APK/AAB). The app provides an interactive quiz experience with age-based questions, instant feedback, score calculation, and an attractive user interface.

## Features

- User name and age registration
- Age-based quiz categories
- Multiple-choice questions
- Instant answer validation
- Automatic score calculation
- Animated celebration effects
- Restart quiz option
- Android APK support

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Android APK/AAB Packaging
- Git & GitHub

## How to Run

1. Install the APK on an Android device.
2. Open the application.
3. Enter your name and age.
4. Start the quiz.
5. View your final score.
